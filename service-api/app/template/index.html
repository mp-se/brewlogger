Brew logger

Welcome to brewlogger, a tool for collecting logging data from GravityMon, iSpindle and KegMon software.

I built this project since I needed to collect data from my devices and use that to analyse their performance. Its built using a standard REST API for managing data with a Web/Javascript frontend to display data.
